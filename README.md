# Fitness Tracker

## About
**Fitness Tracker** is an application designed to monitor and manage your fitness journey. It includes features for workout logging, progress tracking, and personalized fitness insights.

## Tech Stack
- **Frontend:** React.js
- **Backend:** Node.js

## Features
- Log workouts and exercises
- Track fitness progress over time


## Getting Started

### Prerequisites
Make sure you have the following installed:
- [Node.js](https://nodejs.org/) (v14 or higher)
- [npm](https://www.npmjs.com/) (Node Package Manager)

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/N-Diamant/Fit-tracker.git
   cd Fit-tracker
### Running the Application

You can start the application in both the frontend and backend:

For Frontend
Navigate to the frontend directory:
cd frontend

Start the frontend server:
npm start

For Backend
Navigate to the backend directory:
cd ../backend

Start the backend server:
npm start
